# MINECONECT

**MINECONECT** es la plataforma digital líder que conecta, potencia y dinamiza a los actores estratégicos del sector minero bajo una visión de sostenibilidad ambiental.

### 🚀 Conectamos el ecosistema minero
Nuestra plataforma facilita la interacción entre cuatro pilares fundamentales:

| 🏢 **Empresarios** | 🚀 **Emprendedores SENA** | 💰 **Inversionistas** | 🏛️ **Instituciones** |
| :--- | :--- | :--- | :--- |
| Buscan optimizar sus empresas y encontrar socios estratégicos. | Con proyectos innovadores en busca de financiación o mentoría. | En busca de proyectos prometedores y rentables para invertir. | Ofrecen programas, convocatorias y transferencia de conocimiento. |
